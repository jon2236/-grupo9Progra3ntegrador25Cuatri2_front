<h1 align="center">TP INTEGRADOR - Programación 3 (Frontend)</h1>

Descripción
-----------
Proyecto integrador del cursado de Programación 3 orientado al desarrollo frontend. Aplicación simple que demuestra conceptos vistos en clase: componentes, rutas, consumo de APIs y estilos responsivos.

Objetivo
--------
Crear una interfaz funcional y presentable que sirva como entregable final del trabajo práctico integrador.

Estructura del repositorio
--------------------------
- /public — archivos estáticos
- /src — código fuente (componentes, estilos, servicios)
- README.md — este archivo


Instalación y ejecución rápida
-----------------------------
1. Clonar el repositorio
2. Instalar dependencias: npm install
3. Ejecutar en modo desarrollo: npm start
4. Abrir en el navegador: http://localhost:3000
